#### 安装教程

1. 克隆到本地后idea打开项目
2. 在本地MySQL中创建名称为book的数据库，并运行本项目sql目录下的book.sql文件
3. 修改src/main/resources/application.properties的数据库用户名密码为你自己的
4. idea中配置tomcat上下文为空（如下图所示）
